# BACKLOG

案件の一覧とステータスを管理する。

## ステータス定義

- **Open**: 案件作成済み、着手前
- **In Progress**: 作業中(調査・ドキュメント作成・実装・テストのいずれか)
- **Closed**: 完了(手動テスト合格、ドキュメント更新済み)
